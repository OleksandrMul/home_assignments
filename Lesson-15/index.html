ДЗ №15

  • Задача 1

    Розробити клас «Керівник танців»

    Поля

    1. Масив імен хлопців
    2. Масив імен дівчат

    Методи

    1. Метод випадкового вибору імені хлопця
    2. Метод випадкового вибору імені дівчини
    3. Метод виведення пари для танців
    4. Метод run , який ініціює через кожні 5 секунд виведення нової пари для танців

  • Задача 2

    Розробити клас Banner

    Поля

    1. Масив об’єктів ( графічних зображень та посилань на сайти)

    Методи

    1. Метод випадкового вибору об’єкта (графічного зображення та посилання)
    2. Метод виведення випадкового банера

  • Задача 3

    Розробити клас MultChecker для перевірки таблиці множення

    Поля

    1. Число, яке перевіряємо (наприклад, перевірка частини таблиці множення на 7)
    2. Кількість правильних відповідей
    3. Кількість неправильних відповідей

    Методи

    1. Генерування прикладу (метод випадковим чином визначає друге число, перше число фіксоване)
    2. Перевірка правильності вказаної відповіді
    3. render - виведення інформації про тестування на екран
